# empirical-project

# Twitter/X Stock Volatility Analysis (2013-2022)

This project analyses how trading volume and key corporate events surrounding Elon Musk's takeover of Twitter/X affected and predicted next-day stock volatility from the IPO in 2013 to day the company was taken private in 2022.

# Data

The dataset used is the daily stock price for Twitter across the full period of it being a public company from 2013 to 2022, sourced from Kaggle. The link for the data is https://www.kaggle.com/datasets/maharshipandya/twitter-stocks-dataset/data. It includes open price, close, high, low, adjusted close and trading volume columns

# Libraries

Python was used for the code, with the following libraries installed:
- pandas
- numpy
- matplotlib
- statsmodels
- scikit-learn

# Replication

In order to replicate, you have to clone the respository, ensure the folder structure is intact, then open the code in Jupyter notebook. All outputs should be generated automatically.
